#### To run app on local machine:
```bash
$ git clone https://github.com/PriyanshBordia/The-MilkyWay-Airlines.git
$ python3 -m venv venv
$ source venv/bin/activate
$ cd The-MilkyWay-Airlines
$ pip install -r requirements.txt
$ sh run.sh 1
```
