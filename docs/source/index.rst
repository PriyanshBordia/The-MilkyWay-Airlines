Installation
************
   First make sure you have Python 3.8 or higher installed. You can download Python |download_python|.

.. |download_python| raw:: html

To run this project on local machine:

.. code-block:: bash

   $ git clone https://github.com/PriyanshBordia/The-MilkyWay-Airlines.git
   $ cd The-MilkyWay-Airlines
   $ pip freeze > requirements.txt
   $ sh run.sh 1
